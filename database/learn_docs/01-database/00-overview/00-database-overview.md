# 数据库技术概述

## 数据库分类

### 1. 关系型数据库 (RDBMS)

使用 **SQL** 语言，基于 **关系模型**（二维表）存储数据。

**特点**:
- 数据结构严格，需要预先定义表结构
- 支持 **ACID** 事务特性
- 数据关联通过 **外键** 建立
- 适合复杂查询和数据分析

**代表产品**:
| 数据库 | 特点 |
|--------|------|
| MySQL | 最流行的开源数据库，生态丰富 |
| PostgreSQL | 功能最强大的开源数据库 |
| Oracle | 企业级数据库，功能全面 |
| SQL Server | 微软企业级数据库 |

### 2. 文档数据库 (Document Store)

以 **JSON/BSON** 格式存储数据，灵活的结构设计。

**特点**:
- 无需预先定义 Schema
- 适合敏捷开发和迭代
- 支持复杂嵌套数据结构
- 水平扩展能力强

**代表产品**:
| 数据库 | 特点 |
|--------|------|
| MongoDB | 最流行的文档数据库 |
| CouchDB | Apache 项目，Erlang 实现 |
| RethinkDB | 实时推送，支持 JavaScript |

### 3. 键值存储 (Key-Value Store)

最简单的数据库形式，以 **键值对** 形式存储数据。

**特点**:
- 极高的读写性能
- 简单易用，API 简洁
- 适合缓存和高频访问场景
- 一般不支持复杂查询

**代表产品**:
| 数据库 | 特点 |
|--------|------|
| Redis | 支持多种数据结构，内存数据库 |
| LevelDB | Google 开源，嵌入式 |
| RocksDB | Facebook 开源，基于 LevelDB |
| DynamoDB | AWS 托管服务 |

### 4. 列式存储 (Column Family)

按列存储数据，适合大规模数据分析。

**特点**:
- 列式存储，压缩率高
- 适合 OLAP 查询
- 写入性能优秀
- 分布式架构

**代表产品**:
| 数据库 | 特点 |
|--------|------|
| HBase | Hadoop 生态，分布式 |
| Cassandra | 去中心化，高可用 |
| ClickHouse | 列式存储，分析型 |
| Doris | 实时分析数据库 |

### 5. 图数据库 (Graph Database)

以 **图结构** 存储数据，适合关系网络分析。

**特点**:
- 高效处理复杂关系
- 支持图遍历和路径查询
- 适合社交网络、推荐系统
- 专业领域应用

**代表产品**:
| 数据库 | 特点 |
|--------|------|
| Neo4j | 最流行的图数据库 |
| JanusGraph | 分布式图数据库 |
| ArangoDB | 多模型数据库 |

## 数据库选型指南

### 场景选择

| 场景 | 推荐数据库 |
|------|------------|
| 传统业务系统 | MySQL / PostgreSQL |
| 内容管理 CMS | MongoDB |
| 实时分析 | ClickHouse / Doris |
| 高频缓存 | Redis |
| 日志存储 | Elasticsearch |
| 嵌入式存储 | LevelDB / RocksDB |
| 社交关系 | Neo4j |

### 选型考虑因素

1. **数据模型** - 结构化 vs 非结构化
2. **一致性要求** - 强一致 vs 最终一致
3. **扩展性** - 垂直扩展 vs 水平扩展
4. **性能需求** - 读多 vs 写多
5. **运维成本** - 托管 vs 自建
6. **生态成熟度** - 社区支持、工具链

## 本项目涵盖的数据库

### 关系型数据库

#### MySQL
- 最流行的开源关系型数据库
- 丰富的生态和工具链
- 适合 Web 应用和业务系统

#### PostgreSQL
- 功能最强大的开源数据库
- 支持高级 SQL 特性（CTE、窗口函数）
- JSON/JSONB 原生支持

### NoSQL 数据库

#### MongoDB
- 文档型数据库
- 灵活的 Schema 设计
- 强大的聚合管道

### 键值存储

#### LevelDB
- Google 开源的嵌入式键值存储
- 高性能，单线程 Compaction
- 适合学习键值存储原理

#### RocksDB
- Facebook 开源的嵌入式键值存储
- 基于 LevelDB，多线程 Compaction
- 支持列族，性能更高
- 生产环境推荐

## 学习路径

```
数据库学习路线
├── 阶段1: 关系型数据库
│   ├── MySQL 基础（SQL、索引、事务）
│   └── PostgreSQL 进阶（CTE、JSON）
│
├── 阶段2: NoSQL 数据库
│   ├── MongoDB（文档型）
│   └── Redis（缓存）
│
└── 阶段3: 嵌入式存储
    ├── LevelDB（入门）
    └── RocksDB（生产环境）
```
